## The 30-Second Version

You're running late to a meeting. You grab your phone, message Claude: *"Export the Q1 pitch deck as PDF and attach it to my 2 PM calendar invite."* You walk in. It's done.

That's the headline. Claude can now **use your Mac like a human would** — opening apps, navigating browsers, filling spreadsheets, managing files — while you're away. Paired with the new **Dispatch** feature, you assign tasks from your phone and come back to finished work on your desktop.

---

## What's Actually New This Week

Anthropic shipped two tightly integrated features:

### 1. Claude Computer Use (in Cowork)
Claude can now operate your Mac autonomously:
- Open applications
- Browse the web
- Read and edit files and spreadsheets
- Run developer tools (if applicable)
- Navigate desktop UI — no setup or scripting required

It runs locally on your machine, so it has access to your actual files — not just what you upload to a chat window.

**Key safeguard:** Claude asks permission before accessing any new app it hasn't used before. You're always in control.

### 2. Dispatch (Claude Cowork)
Think of Dispatch as your remote control for Claude on your desktop:
- Assign tasks from your iPhone while you're away from your desk
- Claude executes on your Mac in the background
- Return to completed work, no babysitting required

## How to Set It Up

1. **Open Claude.ai** → make sure you're on **Pro or Max** plan
2. Navigate to **Cowork** in the sidebar
3. Enable **Computer Use** — Claude will request macOS accessibility permissions
4. Install the **Claude mobile app** and enable **Dispatch**
5. Start with a low-stakes test: *"Open a new text file and type 'hello world.'"*
6. Once comfortable: go wild with the BA tasks above

---

## The Honest Caveat

Anthropic themselves said it: *"Computer use is still early."* Claude can make mistakes — misclick, misread a UI element, take a wrong turn. **Don't set it loose on anything irreversible without supervision** until you've built trust. Use it first on read/research tasks, then graduate to write/submit tasks.

Think of it like a very smart new hire: capable, eager, but worth a double-check on their first week.
